# SQL-Based Dynamic Pricing Intelligence System

## Overview
A MySQL-based project that analyzes sales data and product demand to generate dynamic pricing recommendations.

## Features
- Product Sales Analysis
- Demand Classification
- Dynamic Pricing Recommendations
- Monthly Sales Trends
- SQL Views and Aggregations

## Technologies Used
- MySQL
- SQL

## Project Workflow
Products Data → Sales Analysis → Demand Analysis → Pricing Recommendation

- Power BI Dashboard
